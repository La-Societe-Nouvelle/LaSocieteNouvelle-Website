## Introduction

L'application web « METRIZ » est mise à disposition pour fournir une solution de base pour la mesure de l'Empreinte Sociétale d'une Entreprise.

Elle fait le lien entre les écritures comptables, l'analyse réalisée lors de l'exercice précédent, les données disponibles pour les fournisseurs et les impacts directs de l'entreprise.

La version locale n'est plus maintenue ! Une version simplifiée est prévue pour l'année 2025.

### Accéder à l'application 

L'application est accessible à l'adresse suivante : [https://partners.metriz.lasocietenouvelle.org](https://partners.metriz.lasocietenouvelle.org)
