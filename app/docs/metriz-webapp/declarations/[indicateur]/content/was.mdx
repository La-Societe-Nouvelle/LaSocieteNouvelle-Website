## Déclaration

[Lien vers la documentation de l'indicateur](/docs/empreinte-societale/indicateurs/was)

### Formulaire de déclaration

La déclaration concerne la production directe de déchets (en kg) et l'incertitude relative. Par défaut, l'incertitude est de 25%.

### Sources de l'information

Liste des documents utiles :

- Factures de traitement des déchets

Pour les déchets assimilés aux ordures ménagères, la production moyenne est de 130 kg/an/personne.
